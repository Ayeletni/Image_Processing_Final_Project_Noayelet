הוראות העלאה קצרות לגיטהאב

1. חלצי את קובץ ה-ZIP.
2. בגיטהאב בחרי Add file ואז Upload files והעלי את תיקיית images בשלמותה.
3. פתחי את README.md שבמאגר, לחצי על סמל העיפרון, מחקי את התוכן הקיים והדביקי את תוכן README.md מהחבילה.
4. שמרי באמצעות Commit changes.
5. את קובצי Google Colab נוסיף בהמשך לתיקיית notebooks.

הערה: התמונות בחבילה הן האיורים והגרפים החתוכים בלבד, ללא מסכי דפדפן או קוד של Colab.
